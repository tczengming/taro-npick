# taro-npick
taro number pickerview, 嵌入页面的数字条滚动选择器

<Prev>
可多列组合，比如时间选择器，则是两列, min, max指定范围，step为步进大小，init为初始值
    data: [
      {
        min: 0,
        max: 23,
        step: 1,
        init: 0,
      },
      {
        min: 0,
        max: 59,
        step: 1,
        init: 0,
      },
    ],
</Prev>
